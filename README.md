# ME310FarmAssistant

for ME310 Course

Developed by a group of hydraulicians...: )

## Usage

```bash
$ git clone https://github.com/h1994st/ME310FarmAssistant.git
$ cd ME310FarmAssistant
$ pod install
$ open ME310FarmAssistant.xcworkspace
```
